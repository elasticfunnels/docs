Clickbank integration for affiliates automatically tracks conversions and postbacks.

1. Go to "Settings" or "Integrations"
2. Click "Activate" and provide the API key (see How to Create ClickBank API Keys)

## Creating ClickBank API Keys

1. Log into your ClickBank account
2. Go to Settings â†’ My Account
3. Click on "API Keys" section
4. Generate a new API key with appropriate permissions
5. Copy the API key for use in the integration

## Integration Benefits

- **Real-time Tracking**: Instant notification of sales and commissions
- **Automated Reporting**: Automatic sync of performance data
- **Commission Management**: Track earnings across all your promotions
- **Refund Handling**: Automated processing of refunds and chargebacks

<Note>This integration is used as an affiliate. To set up ClickBank as a merchant, see [ClickBank](/merchants/clickbank).</Note>

## API Key Permissions

Ensure your API key has the following permissions:
- Read access to order data
- Access to commission information
- Refund and chargeback data access

## Security Best Practices

- Keep your API key secure and never share it
- Regularly rotate API keys for security
- Monitor API usage for any unusual activity
- Use the minimum required permissions for your needs
